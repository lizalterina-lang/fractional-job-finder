# Fractional Job Finder

## Описание

Интерактивный каталог стартапов и скейлапов из Австралии, Сингапура и Великобритании, где можно найти удалённую fractional работу в роли PMM / GTM / Growth на 4–6 часов в день. Приложение помогает быстро фильтровать компании по стране, роли и ключевым словам, а также найти подходящие платформы для поиска и тактики для получения первых клиентов.

## Возможности

- **Каталог компаний** — 30 стартапов и скейлапов из AUS, SG и UK с описанием, нишей, стадией и ссылками на вакансии
- **Фильтрация по стране** — Австралия 🇦🇺, Сингапур 🇸🇬, Великобритания 🇬🇧
- **Фильтрация по роли** — PMM, GTM, Growth
- **Поиск** — мгновенный поиск по названию компании, нише и описанию
- **Платформы для поиска** — 12 специализированных и общих платформ для поиска fractional работы
- **Тактики аутрича** — 6 конкретных тактик для получения первых клиентов
- **Прямые ссылки** — на страницы вакансий и официальные сайты компаний
- **Адаптивный дизайн** — работает на мобильных устройствах
- **Живые вакансии** — парсер `scraper.py` загружает актуальные вакансии с Remote OK, WorkingNomads и LinkedIn (RapidAPI); запуск: `python3 scraper.py`
- **Telegram-дайджест** — `send_telegram.py` отправляет ежедневный дайджест в Telegram канал; запуск: `python3 send_telegram.py`
- **Ежедневный автозапуск** — `daily_run.py` объединяет парсинг и отправку; добавь в cron для автоматики
- **Watchlist компаний** — 40+ компаний AUS/SG/UK/NZ из watchlist всегда помечаются ⭐ в дайджесте
- **Все настройки** в `.env` файле: Telegram токен и chat_id, RapidAPI ключ
